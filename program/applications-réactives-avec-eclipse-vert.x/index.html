Applications réactives avec Eclipse Vert.x

Julien Viet, lead du projet Vert.x, présente ce toolkit pour écrire des applications réactives et polyglottes sur la JVM.

La session commence par une vue d’ensemble de Vert.x et des paradigmes de programmation réactive, puis introduit la programmation réactive avec RxJava 2 et Vert.x.

Points abordés

  • Architecture event-loop de Vert.x
  • Verticles et event bus
  • Programmation réactive avec RxJava 2
  • Cas d’usage : APIs REST, microservices, streaming
