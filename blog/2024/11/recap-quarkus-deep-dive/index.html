Recap : Quarkus Deep Dive

recap quarkus cloud-native

Une soirée placée sous le signe du cloud-native

Le 19 novembre dernier, plus de 40 développeurs se sont réunis au Parc Belair pour notre session Quarkus Deep Dive. Jane Doe nous a offert une présentation complète et très appréciée du framework Quarkus.

Ce qu’on a appris

Quarkus en bref

Quarkus est un framework Java pensé pour les environnements cloud-native et Kubernetes. Ses points forts :

  • Démarrage ultra-rapide (quelques millisecondes)
  • Empreinte mémoire réduite
  • Rechargement à chaud en développement
  • Mode natif via GraalVM

Live coding

Jane a réalisé une démonstration live de la création d’un microservice REST avec Quarkus, en montrant le mode dev et la compilation native. Le contraste avec une application Spring Boot classique était saisissant.

Merci !

Un grand merci à Jane pour sa présentation, et à tous les participants pour leur enthousiasme.
