## Ehcache 3: JSR-107 survitaminé

Louis Jacomet et Ludovic Orban présentent **Ehcache 3**, la nouvelle version de la bibliothèque de cache Java la plus populaire.

### JSR-107 : l'API de cache standard
- Présentation de JSR-107, l'API de cache temporaire pour la plateforme Java
- Intégration facile avec Spring et autres frameworks

### Ehcache 3 : au-delà du standard
- Refonte majeure de l'API
- **Scalabilité verticale** : stockage off-heap pour dépasser les limites du heap JVM
- **Scalabilité horizontale** : clustering et distribution du cache
- Support transactionnel
- Write-behind et write-through
