## Des containers de la cale au pont

Nicolas Muller présente **Docker** et son écosystème pour les développeurs Java.

La session de 90 minutes inclut des démonstrations live orientées vers les cas d'usage du développement quotidien.

### Au programme
- Docker : concepts fondamentaux
- Docker Compose — orchestration locale
- Docker Machine
- Docker Swarm
- Patterns architecturaux pour applications Java
- Comment tirer le meilleur parti de Docker dans votre travail Java quotidien
