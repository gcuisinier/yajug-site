## 3 Idées d'amélioration pour vos Architectures Microservices

Nicolas Frankel (Hazelcast) présente des solutions concrètes pour améliorer les performances des architectures en microservices.

### Contexte
Une architecture microservices est plus évolutive qu'une approche monolithique, mais elle a un impact direct sur les performances. Chaque appel réseau entre services a un coût.

### Les 3 idées
1. **Cache distribué** pour les accès base de données avec Hazelcast IMDG
2. **Cache des invocations REST** entre microservices
3. **Gestion d'état de session** dans les clusters serveur
