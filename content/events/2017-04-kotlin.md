## Kotlin

Kotlin est un langage basé sur la JVM développé par **JetBrains** depuis 2010, inspiré de Java, Scala, C# et Groovy.

La présentation couvre :
- Syntaxe du langage et ses avantages
- Interopérabilité avec Java
- Intégration dans des stacks Spring Boot
- Null safety, data classes, extensions
- Coroutines

Depuis mai 2017, Kotlin est officiellement supporté pour le développement Android.
