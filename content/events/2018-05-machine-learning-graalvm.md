## Déployer facilement vos services de machine learning en Java avec GraalVM

Philippe Gottfrois et Pierre Paci explorent comment déployer des solutions de machine learning au sein d'infrastructures Java existantes.

Ils démontrent comment **GraalVM**, le projet open source d'Oracle, permet d'intégrer des segments de code Python dans des applications Spring, ouvrant la voie à une synergie entre l'écosystème Python ML et le monde Java enterprise.
