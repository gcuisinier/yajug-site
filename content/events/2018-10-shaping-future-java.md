## Shaping the Future of Java, Faster

La plateforme Java SE et le JDK ont adopté un **cycle de releases rapide de 6 mois**, avec Java 11 en septembre 2018 et Java 12 prévu pour mars 2019.

Cette présentation examine les motivations de ce changement et discute de ses implications pour les développeurs, les entreprises et l'écosystème Java en général.

### Sujets abordés
- Motivation du nouveau cycle de release
- Java 11 LTS : ce qui change
- Impact sur les versions Long-Term Support (LTS)
- Ce que cela signifie pour les projets en production
