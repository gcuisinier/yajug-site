## Apache Solr : sans tambour ni trompette

La présentation couvre quatre axes principaux pour maîtriser Apache Solr :

1. **Architecture** — Comment fonctionne Solr sous le capot
2. **Indexation** — Stratégies d'indexation efficaces
3. **Configuration de la recherche** — Scoring, facettes, suggestions
4. **Configuration du cluster** — SolrCloud, sharding, réplication

L'objectif : donner aux participants les clés pour implémenter des capacités de recherche full-text dans leurs projets.
