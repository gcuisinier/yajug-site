## De Zéro à Héros avec Spring Boot

Stéphane Nicoll, core committer Spring Boot, présente comment Spring Boot combine la flexibilité de Spring avec des valeurs par défaut sensées.

### Cas d'usage couverts
- Services REST
- Plateformes web
- Applications batch

### Thèmes principaux
- Auto-configuration et starters
- Développement microservices
- Pratiques DevOps avec Spring Boot
- Actuator et observabilité
