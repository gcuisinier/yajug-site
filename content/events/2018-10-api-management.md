## Et si on se mettait à l'API ?

En 2018, les services cloud exposent leurs données via des APIs. Cette présentation explore les questions clés :

- Comment définir ses APIs avec **OpenAPI** (Swagger) ?
- Quelles bonnes pratiques pour la documentation ?
- Comment sécuriser l'accès ?
- Quelles stratégies de monétisation ?
- Quel rôle pour un API Gateway / API Manager ?
