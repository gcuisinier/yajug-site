## De meilleures performances avec HTTP/2 et programmation réactive avec Vert.x

Julien Viet présente deux sujets complémentaires sur la performance des applications Java modernes.

### HTTP/2 côté serveur
- Impact d'HTTP/2 sur les performances serveur
- Multiplexage des connexions
- Server push
- Pourquoi le modèle de concurrence de Vert.x est particulièrement adapté à HTTP/2

### Programmation réactive avec Vert.x
- Introduction à Eclipse Vert.x
- Le modèle event-loop et ses avantages
- L'écosystème Vert.x : Web, Service Proxy, JDBC, etc.
- Patterns réactifs en pratique
