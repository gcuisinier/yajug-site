## Java is going nuts|native

Rémi Forax (Université Paris Est Marne-la-Vallée) présente **Project Jigsaw** et le système de modules de Java 9.

### Au programme
- Pourquoi Java avait besoin d'une architecture modulaire
- Comment fonctionnent les modules Java 9
- Migration des applications Java 8 vers Java 9
- Problèmes de compatibilité et solutions
- Nouvelles fonctionnalités :
  - Services (ServiceLoader amélioré)
  - Images runtime personnalisées (jlink)
  - Compilation Ahead-of-Time (AOT)
