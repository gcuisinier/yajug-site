## Programme

Jean-Michel Doudoux nous présente un tour d'horizon complet des nouveautés de **Java 21 LTS**.

### Project Amber
- Record Patterns
- Pattern Matching for switch
- String Templates
- Unnamed Patterns and Variables
- Unnamed Classes

### Project Loom
- **Virtual Threads** — la révolution de la concurrence Java
- Structured Concurrency
- Scoped Values

### Project Panama
- Foreign Function & Memory API
- Vector API

### Core API & HotSpot JVM
Améliorations des API centrales et du moteur d'exécution HotSpot.
