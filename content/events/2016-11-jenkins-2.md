## Jenkins 2.0

Adrien Lecharpentier présente les changements majeurs apportés par **Jenkins 2.0** après plus de dix ans de versions 1.x.

### Au programme
- Pourquoi Jenkins passe en version 2 ?
- Les changements apportés par cette nouvelle version majeure
- **Pipeline** : la nouvelle façon de définir vos builds as code
- Aperçu de **BlueOcean** : la nouvelle interface utilisateur de Jenkins
