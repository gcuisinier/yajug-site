---
title: "About Us"
description: "YAJUG is the Java User Group of Luxembourg, organized by passionate volunteers."
---

YAJUG (Yet Another Java User Group) has been bringing Java developers together in Luxembourg
since 2008. We organize regular meetups on Java, JVM technologies, and the broader
software craftsmanship ecosystem.

We are a **non-profit, vendor-neutral** community run entirely by volunteers.
If you'd like to speak at one of our events or get involved, don't hesitate to
[contact us](/contact/).
