Contributeur Jenkins depuis 2009, Adrien a débuté avec son premier plugin Hudson chez Alcatel-Lucent. Consultant CI/CD chez Zenika puis ingénieur support chez CloudBees, il se spécialise aujourd'hui dans les migrations d'infrastructure Jenkins : upgrades Java (8→11→17→21), migration Spring Security 6, et maintenance de l'écosystème de plugins. Committer Apache (Archiva) et Jenkins, il milite pour une meilleure expérience utilisateur et l'accessibilité de Jenkins aux nouveaux contributeurs.
